
```
Compile with:
   go build
   
Usage of ./egg:
  -dst string
    	where to save new egg (default "new_egg.jpg")
  -new string
    	what new egg should look like (default "eegg")
  -src string
    	path for mom egg (default "egg.jpg")
  -template string
    	template for mom egg (default "egg")
```
